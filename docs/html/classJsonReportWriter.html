Writes the list of vulnerability matches to a JSON file.

Writes a list of detected vulnerabilities to a JSON file.

Parameters
vulnsList of matched vulnerabilities.
outputPathPath to the output JSON file.

This function iterates over a vector of MatchResult objects and creates a JSON structure containing relevant information about each vulnerability, such as:

  • File path
  • Line number
  • Pattern ID
  • Message
  • Severity
  • Code line

The final output is formatted and saved to the file specified by outputPath.

Parameters
vulnsA vector of MatchResult entries representing detected vulnerabilities.
outputPathThe path to the output file where the JSON data will be written.
