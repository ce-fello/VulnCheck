Analyzes the specified file against a list of vulnerability patterns.

Analyzes a file line by line using provided vulnerability patterns.

Parameters
filepathPath to the file to analyze.
patternsList of vulnerability patterns to search for.
Returns
std::vector<MatchResult> List of matched vulnerabilities found in the file.

Each line is scanned for both plain-text and regex-based patterns. When a match is found, the result is recorded with associated metadata including file, line number, severity, and code snippet.
